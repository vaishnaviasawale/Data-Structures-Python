PART 1️⃣ — What Else You Should Know (Critical but Subtle)
1. Python Mutability (VERY IMPORTANT)
Type	Mutable?
list	✅
dict	✅
set	✅
tuple	❌
str	❌
Why it matters
def f(x):
    x.append(5)

lst = [1,2]
f(lst)
print(lst)  # [1,2,5]


👉 Can cause unexpected side effects in DFS/BFS, recursion, etc.

2. Shallow vs Deep Copy
a = [[1,2], [3,4]]
b = a[:]          # shallow
c = [x[:] for x in a]  # deep


Needed in:

Graph traversal

Backtracking

DP memoization

3. Recursion Limits
import sys
sys.setrecursionlimit(10**7)


DFS on deep graphs / trees can crash otherwise.

4. Stable vs Unstable Sorting

Python sorted() is stable.

sorted(data, key=lambda x: x.age)


Stability matters in:

Multi-key sorting

Greedy algorithms

5. Hash Collisions & Security

Python randomizes hash seed

Prevents hash-flood attacks

Dict/set worst-case exists but rare

6. When NOT to Use a Data Structure
Need	Don’t Use
Fast search	list
Order matters	set
FIFO	list
Priority	queue
Uniqueness + order	set
7. Big-O Reality Check (Interview Gold)

Big-O ignores constants, but Python does not

Example:

dict lookup faster than binary search

deque faster than list for queue

heapq slower than sorted() for one-time sorting

PART 2️⃣ — Lambda Functions with DSAs (THE RIGHT WAY)
🔹 What is a lambda?
lambda x: x * 2


Anonymous, one-line function.
Used when behavior matters more than structure.

1️⃣ Lambda with list
Sorting
arr = [(1,3), (2,1), (4,2)]
arr.sort(key=lambda x: x[1])

Filtering
even = list(filter(lambda x: x % 2 == 0, arr))

Mapping
squares = list(map(lambda x: x*x, arr))

2️⃣ Lambda with dict
Sort by value
d = {'a':3, 'b':1, 'c':2}
sorted(d.items(), key=lambda x: x[1])

Custom priority
max(d, key=lambda k: d[k])

3️⃣ Lambda with set
Custom sort (convert to list)
s = {3,1,4}
sorted(s, key=lambda x: -x)

4️⃣ Lambda with heapq (VERY IMPORTANT)
Priority Queue with custom priority
import heapq

pq = []
heapq.heappush(pq, (priority, value))


Using lambda to define priority:

tasks = [('A', 3), ('B', 1)]
tasks.sort(key=lambda x: x[1])


For max heap:

heapq.heappush(pq, (-priority, value))

5️⃣ Lambda with DFS / BFS
Sorting neighbors
for nei in sorted(graph[node], key=lambda x: x):
    dfs(nei)

BFS with priority
queue.sort(key=lambda x: x.cost)

6️⃣ Lambda with deque
dq = deque(sorted(data, key=lambda x: x[0]))

7️⃣ Lambda in Graph Algorithms
Dijkstra
heapq.heappush(pq, (dist[node], node))


Custom:

edges.sort(key=lambda x: x.weight)

8️⃣ Lambda vs Function (Interview Tip)

Use lambda when:

Function is used once

Logic is short

Readability improves

Use def when:

Multi-line logic

Reuse required

Debugging needed

🔥 COMMON MISTAKES WITH LAMBDA

❌ Too complex

lambda x: x.a if x.b > 0 else x.c * x.d


❌ Using lambda for recursion (not allowed)

❌ Using lambda when clarity suffers
